## Getting Started

### Installation

Install dependencies:

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

Build for production:

```bash
npm run build
```

## Technologies

- React 18 with Hooks (useState, useEffect, useRef)
- Vite for fast development and building
- CSS3 with advanced animations
- Local Storage API for persistence

## Component Structure

### Navigation
- **NavigationBar**: Sticky navigation bar with page routing

### Pages
- **Dashboard**: Main overview page with TaskManager
- **Tasks**: Dedicated task management page
- **Analytics**: Statistics and insights page
- **Settings**: Application settings and preferences

### Components
- **TaskManager**: Main task management component
- **TaskInput**: Expanding input with priority selector
- **TaskList**: Task container with drag-and-drop
- **TaskItem**: Individual task with glass-morphism card
- **ProgressRing**: Animated circular progress indicator
- **StatisticsPanel**: Animated statistics with bar charts
- **FilterTabs**: Sliding highlight filter navigation
- **ParticleBackground**: Interactive particle effects
